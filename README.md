# Proyecto en IntelliJ IDEA

Este es mi proyecto hecho en *IntelliJ IDEA* con *Java*.  
La idea del trabajo es mostrar cómo funciona un sistema con su estructura, ventanas y parte lógica, hecho paso a paso con lo que vimos en clase.

El programa está organizado por carpetas, con las clases y funciones necesarias para que todo se entienda fácil.  
La intención es que se vea cómo se maneja la información dentro del sistema y cómo se pueden hacer las operaciones básicas (como agregar, mostrar o actualizar datos).

Todo el código está hecho por mí y subido aquí para tenerlo guardado y poder mostrarlo.  
Está listo para funcionar si se configura bien en otro computador.

---

## Tecnologías usadas
- *Java*
- *IntelliJ Idea

Este trabajo lo hice con dedicación y con base en lo aprendido en clase.  
Puede servir como ejemplo o punto de partida para seguir mejorando el sistema más adelante.
